# AI-
• 面向缺少实习经历、不会写简历的大学生群体，梳理「低经验学生简历优化」的核心需求，聚焦经历表达弱、岗位匹配度低、内容不够专业等痛点。 • 通过 Codex 辅助独立搭建 AI 简历生成器，支持用户输入基础经历、求职意向、项目描述等少量文本后，自动生成结构完整、表达更专业、适合投递的简历内容。 • 使用 ChatGPT、Codex、Gemini 进行提示词设计、内容结构优化、生成逻辑调试和功能迭代，提升生成内容的完整度、岗位相关性和表达专业度。 • 围绕「用户输入成本低、生成内容可用、简历修改效率高」设计核心流程，将简历撰写任务拆解为信息采集、经历提炼、岗位匹配、表达优化等步骤。
